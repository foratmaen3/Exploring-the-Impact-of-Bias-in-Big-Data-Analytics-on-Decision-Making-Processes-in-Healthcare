# Big Data Analytics in Healthcare: Bias and Decision-Making

This repository contains a comprehensive study exploring the impact of bias in big data analytics within the healthcare sector. The project aims to identify how biases in data collection and processing affect decision-making processes in healthcare and proposes mitigation strategies to handle these biases effectively.

## Project Structure

The repository is structured as follows:

- `Data/`: This directory contains the raw data files used in the analysis, including survey data and secondary data sources.
- `Docs/`: Documentation for the project, including the project proposal, interim reports, and the final research paper.
- `Notebooks/`: Jupyter notebooks with detailed analysis, including data cleaning, exploratory data analysis, and statistical tests.
- `Scripts/`: Python scripts used for more extensive data processing and analysis tasks.
- `Survey/`: Contains the survey questionnaire and links to the survey results.
- `Results/`: Final results of the study, including charts, tables, and other outputs generated from the data analysis.
